Interactive · ViT patch embedding

An 8×8 image is cut into non-overlapping p×p patches (row-major). Each patch is flattened to a length-C·p·p vector and linearly projected to a D-dim token. A learnable [CLS] token is prepended → sequence of N+1 tokens. Hover a patch to find its token.
