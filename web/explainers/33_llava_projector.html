Image features pass through a 2-layer MLP (GELU) that maps them to the LLM's word-embedding dimension, then the visual tokens are concatenated before the text tokens into one sequence. Hover a stage to see what it does.

h = GELU(x·W1 + b1)·W2 + b2. The projector is applied independently per image token and only changes the channel dim (D_vision → D_text). No attention, no mixing across tokens.
