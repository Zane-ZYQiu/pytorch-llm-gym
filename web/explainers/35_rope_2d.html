Interactive · 2D-RoPE for image tokens

Image tokens sit on an H×W grid (row-major, T = H·W). The head dim is split in half: the first half is rotated by the token's row index, the second half by its col index. Hover a token to see its (row, col) and the two rotation angles.

H=4, W=5 grid — hover a token
θ-pairs. Within each half, dim-pairs use frequencies θm = base−2m/(hd/2). The applied angle for pair m is coord · θm — same RoPE, but the "position" is the row for the first half and the col for the second half.
