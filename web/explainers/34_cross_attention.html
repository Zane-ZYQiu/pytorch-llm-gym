Interactive · cross-modal cross-attention

Text queries (rows) attend to image features (columns). The only change from self-attention: Q comes from the text sequence, while K/V come from the image context — so this is a Tx×Tc grid (no causal mask: a token may look at the whole image). Each row is softmax(QKᵀ/√d) and sums to 1.

self vs. cross.
self-attn: Q, K, V all from one sequence.
cross-attn: Q from text, K/V from image (a different sequence).

Here Tx=4 text queries, Tc=6 image keys, d=4. Each text token retrieves a weighted blend of image V-vectors.
